### 项目描述

鉴于杭州师范大学又被称为杭州吃饭大学，却因为没有一个像样的杭师大美食介绍网站系统，而让很多人错过了新出的美食，以及在地理上毫无规律可寻的食堂位置，导致很多人不知道该去哪里品尝杭师大的美食。
故此我们为杭师大仓前校区所有食堂以及其周边美食设计了一个网站，主要功能用于展示美食、介绍食堂、对食堂菜色评价、阿姨打分、外卖服务等。
